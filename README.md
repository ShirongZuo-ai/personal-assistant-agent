# Personal Assistant Agent

A modular personal AI assistant prototype based on Retrieval-Augmented Generation, tool calling, and LLM agents.

This project explores how large language models can be used to build a personal AI assistant that supports task planning, document understanding, retrieval-augmented generation, and tool-based execution.

The long-term goal of this project is to develop a reliable AI assistant that can understand user goals, break down complex tasks, retrieve useful information, call external tools, and generate actionable responses.

---

## Overview

Personal Assistant Agent is designed as a lightweight experimental framework for building LLM-based agents.

The current version focuses on a basic but clear agent workflow:

1. A user provides a task.
2. The Planner Agent decomposes the task into executable steps.
3. The Executor Agent executes or simulates the planned steps.
4. External tools can be called when needed.
5. The system returns a structured response.

This repository is also intended to serve as a foundation for future integration with stronger LLM APIs, long-context models, RAG pipelines, and multi-agent collaboration frameworks.

## Core Modules

### Planner Agent

The Planner Agent is responsible for analyzing a user task and breaking it into smaller executable steps.

Example responsibilities:

- Understand the user goal
- Identify required information
- Decompose the task into steps
- Decide whether tools or retrieval are needed
- Prepare a structured execution plan

### Executor Agent

The Executor Agent is responsible for executing the plan generated by the Planner Agent.

Example responsibilities:

- Follow the generated task plan
- Execute each step in order
- Call tools when necessary
- Collect intermediate results
- Generate a final response

### Tool Calling

The tool module demonstrates how an agent can call external functions.

The current version includes a simple calculator tool as a minimal example. In the future, this module can be extended to support file operations, web APIs, calendar tools, local scripts, and other external capabilities.

### RAG Module

The RAG module is designed for document-based question answering.

Planned components include:

- Document loading
- Text splitting
- Simple retrieval
- Embedding-based retrieval
- Vector database integration
- LLM-based answer generation

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/ShirongZuo-ai/personal-assistant-agent.git
cd personal-assistant-agent
```

### 2. Create a virtual environment

```bash
python -m venv .venv
```

On Windows:

```bash
.venv\Scripts\activate
```

On Linux or macOS:

```bash
source .venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the demo

```bash
python main.py
```
